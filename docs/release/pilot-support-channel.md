# Pilot support/contact channel

## Kanal

Task 408 definerer support og kontakt for pilot. Konkret kanal for pilot er GitHub Issues i dette repoet med prefiks `[pilot-support]` i tittelen. Kritiske offline-/personvernfeil merkes i tittelen og rutes til pilotansvarlig før bredere pilot.

## Mal for melding

- Tittel: `[pilot-support] Kort problemnavn`
- Miljø: staging eller produksjon, nettleser/enhet og innholdsversjon.
- Reproduksjon: steg med demooppdrag eller anonym testdata.
- Forventet/faktisk resultat.
- Skjermbilde bare hvis det er sladdet og uten persondata.

## Dataregler

Ikke persondata, ikke private posisjoner, ikke ekte hendelsesdetaljer. Bruk skjermbilder med sladding eller demooppdrag.

## Felles grense

Beredskapsboka er beslutningsstøtte, ikke et offisielt kommando-, journal- eller varslingssystem. Ikke skriv inn persondata, private posisjoner, reelle Nødnettgrupper eller annen skjermet informasjon. Appen har ingen backend-sync i MVP; lokal ordre, samband og offisielle kilder gjelder alltid.
