# Short video / interactive guide plan

## Timing

Task 406 er planlagt after UI stabilizes / etter UI stabiliseres. Ikke spill inn persondata eller ekte hendelsesdata. Bruk demooppdrag, offentlig innhold og testkonto/enhet.

## Første moduler

- Kort video: installer til hjemskjerm og offline-test.
- Interaktiv guide: opprett lokal øvelse, kjør sjekkliste, eksporter anonym status.
- Sikkerhetsintro: appen er beslutningsstøtte, ikke offisiell ordre.

## Felles grense

Beredskapsboka er beslutningsstøtte, ikke et offisielt kommando-, journal- eller varslingssystem. Ikke skriv inn persondata, private posisjoner, reelle Nødnettgrupper eller annen skjermet informasjon. Appen har ingen backend-sync i MVP; lokal ordre, samband og offisielle kilder gjelder alltid.
